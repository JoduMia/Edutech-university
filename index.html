Both flex and grid are the pros and cons for modern website design. Although there have not much difference between them, they have some dimensional difference. Flex is one dimensional but grid are two dimensional.

Both are powerful css framework. But tailwindcss maintain Utility first approach to design a layout. On the other hand Bootstrap is component base framework. It's need to write a lot of custom css to design a layout which is really boring. On the other hand, with tailwindcss you can easily make a design pixel perfect without giving a single hand to a custom css file. it has dark mode feature. Finally to say, I love tailwindcss although i am doing the assignment with bootstrap. Tailwind is just beauty!

Semantic tag means those HTML tags which has a realistic inherent meaning. Developer can easily pick what's the purpose of a semantic tag in website. Example, (header,footer,article,main etc)
